# Vagrant Box for TYPO3

## Software

### Server

* Apache

### Database

* MySQL 5.6

### PHP

* PHP 7.0
  * gd
  * MySQL
  * SOAP
  * XML
  * zip

### TYPO3

* TYPO3 8.7.4

### Image Libraries

* GraphicsMagick
* Imagemagick

## Usage

Start Vagrant box

```
vagrant up
```

Stop Vagrant box

```
vagrant halt
```

## Access

Front-End: [http://localhost:8080/](http://localhost:8080/)  
Back-End: [http://localhost:8080/typo3](http://localhost:8080/typo3)
