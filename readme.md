# Vagrant Box for TYPO3

## Usage

Start Vagrant box

```
vagrant up
```

Stop Vagrant box

```
vagrant halt
```

## Access

Front-End: [http://localhost:8080/](http://localhost:8080/)  
Back-End: [http://localhost:8080/typo3](http://localhost:8080/typo3)
